# PetHaven

Plataforma tipo Airbnb para mascotas. Encuentra cuidadores verificados cuando viajas.

## Despliegue en Netlify

1. Entra en [netlify.com](https://netlify.com) y conecta tu cuenta de GitHub.
2. Selecciona el repositorio `pethaven`.
3. Netlify detectará automáticamente la configuración de `netlify.toml`:
   - **Publish directory:** `public`
   - No requiere comando de build
4. Haz clic en **Deploy site**.

Tu landing quedará desplegada en una URL como `https://random-name-123.netlify.app`. Puedes configurar un dominio personalizado en la configuración del sitio.
